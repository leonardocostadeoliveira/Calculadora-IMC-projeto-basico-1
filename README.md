Esse projeto é uma calculadora de IMC simples, desenvolvida com foco nos fundamentos de HTML, CSS e JavaScript.
O objetivo é demonstrar a interação dinâmica, a estrutura responsiva e o uso de eventos em JavaScript." Assim, fica mais completo!

Instrução de uso:

"Instruções de uso: 
Primeiro, insira seu peso no campo correspondente. Em seguida, digite sua altura em metros, utilizando ponto e não vírgula. Após isso, clique no botão 'Calcular'. 
A calculadora exibirá o índice de massa corporal (IMC) e classificará o resultado em quatro níveis: abaixo do peso, peso ideal, acima do peso ou obesidade mórbida, 
com base nas diretrizes de saúde.
